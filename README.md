# Rusty Pong

RustyPong is a basic implementation of old-school Pong using Rust.

## Installation

cargo install

## Usage

Build: cargo build  
Quick build & Run: cargo run  

##Controls

Player 1: WASD  
Player 2: Directional arrow keys 
Space Bar to start the game
