# Rusty Pong

RustyPong is a basic implementation of old-school Pong using Rust.

## Installation

cargo install

## Usage

Build: cargo build  
Quick build & Run: cargo run  

##Controls\n\n

Player 1: WASD\n\n
Player 2: Directional arrow keys\n\n
